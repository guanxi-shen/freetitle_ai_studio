# Turnaround Guide

## Default Format

2x2 grid reference sheet with 4 views:
- Top left: Front view (facing camera)
- Top right: 3/4 angle view
- Bottom left: Side profile (90-degree)
- Bottom right: Back view

## Prompt Structure

Build as a single narrative paragraph: layout instructions, character description (appearance, clothing, features), style (art style, color palette), background (neutral/clean), constraints ("no text overlays, consistent lighting across all views").

## Consistency Across Angles

Same lighting direction, identical outfit/proportions, consistent skin/hair/eye color, neutral expression unless specified, clean background separation.

## Iterative Refinement

Include original as reference image. State "maintaining character design from reference" and describe only what should change.

## Format Adaptation

2x2 is default. Adapt when requested: 3-view strip, 6-view sheet, expression sheet, action pose sheet.

## Exact Replication

For characters matching a reference (real people, brand mascots): minimal description, "exactly as shown in reference image." Let the reference drive the visual -- prompt adds only layout and style.
