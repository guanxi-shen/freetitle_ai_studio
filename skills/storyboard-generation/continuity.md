# Continuity Principles

Visual consistency across shots within a scene and across the entire project.

## Style Guide Application

The `style_guide` from `production_notes` is the primary visual DNA. Every frame in the project should reflect it:
- Color palette and grading approach (teal-orange, bleach bypass, etc.)
- Lighting philosophy (natural, dramatic, high-key, low-key)
- Camera language (focal length preferences, framing conventions)
- Texture and grain (film stock references, digital clean, analog artifacts)

Read the style_guide before writing any prompts. Weave its elements into every frame prompt rather than treating it as an afterthought.

## Within-Scene Consistency

Shots within the same scene share physical reality:
- Same time of day (lighting direction, color temperature, shadow length)
- Same weather and atmosphere (fog density, rain intensity, sky condition)
- Same location details (architecture, furniture, props placement)
- Same wardrobe and character appearance

When writing prompts for sequential shots in one scene, maintain these constants. Vary only what the script dictates: camera angle, shot size, subject focus.

## Across-Scene Consistency

Characters and recurring locations should feel like the same world:
- Character appearance matches turnaround reference (hair, skin tone, outfit)
- Recurring locations maintain architectural identity (same building, same room layout)
- Color grade stays within the style_guide palette (scene mood can shift within the palette, not outside it)

## Lighting Coherence

The most common continuity break. Check:
- Light source direction matches across all shots in a scene
- Color temperature consistent (all 3200K warm, or all 5600K daylight -- not mixed unless motivated)
- Shadow direction and intensity match the established light
- Practical lights (lamps, screens, fires) contribute consistently

## Prompt Techniques for Consistency

- Reference the same camera/lens spec across a scene: "35mm lens, f/2.8" in every shot prompt
- Use the same color grading terms: "teal-orange grade, crushed blacks, warm highlights"
- Name the lighting setup once and repeat: "Rembrandt lighting, key from upper right"
- Include consistent atmosphere: "light haze, golden hour warmth, dust motes"

## When Consistency Should Break

Intentional discontinuity for creative effect:
- Time jumps (color shift to indicate past/future)
- Dream sequences (different texture, different color palette)
- Perspective shifts (different visual treatment per character POV)
- Escalation (shots get progressively more saturated, closer, more intense)

These breaks should be deliberate and motivated by the script, not accidental.
